Статусы партнеров по платежной дисциплине (ТАБ АИД) – используются для классификации платежеспособности клиентов. 
Заполняется в карточке партнера (реквизит – Статус платежной дисциплины).

Заполнение типовыми элементами справочника производится в пункт меню: Автоматизация информирования дебиторов – Статусы партнеров по платежной дисциплине (ТАБ АИД)

Нажимаем кнопку Заполнить, в список добавляются типовые элементы справочника

Для создания нового элемента справочника нажимаем кнопку Создать в открывшемся окне заполняем Наименование далее нажимаем кнопку Записать и закрыть
